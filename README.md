医学诊断标注：
压缩超声（Compression Ultrasonography, CUS）的诊断原理——深静脉血栓的金标准影像诊断之一。医生用超声探头轻压静脉，正常静脉会"消失"（塌陷），而有血栓的静脉会拒绝塌陷，始终可见。


“静脉不能被探头压瘪” 是下肢深静脉血栓（DVT）彩色多普勒超声（也叫压缩超声或加压超声）检查中最重要、最可靠的核心诊断征象，其敏感性和特异性在近端深静脉（股静脉）可达95%以上，甚至被视为金标准级别的单项指标。


基于超声视频的深静脉血栓的自动检测

数据集：

DVT数据集，用于SAM2的分割。
train300例（全是正常人样例）、val76例（38例是正常人样例，38例是患者样例）。
每个case中含有
images文件夹（00000.jpg起递增，每一帧都有）
masks文件夹（00000.png一定有，稀疏标注，只含像素0（背景）、1（动脉）、2（静脉））


YOLO数据集，用于YOLO的训练。
含有每一帧医生标注的图像images及其labels（yolo格式）
train、val所利用的样例和DVT数据集一致

任务：
先训练一个YOLO模型，确保提供每个样例第一帧中静脉box、动脉box。再做些数据增强，置信度调整，只取两个框，一个动脉，一个静脉。若只检测出其中一个，可以通过相对位置等手段生成另外一个。

YOLO提供检测框作为prompt输入给SAM2来完成精确分割，主要利用记忆模块！

分割后通过二分类相关算法，实现检测判断、是否患病。
